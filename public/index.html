Space Shuttle Orbiters

Get info on Atlantis, Challenger, Columbia, Discovery, Endeavour, and Enterprise

Request

GET https://nasa-orbiter-api.herokuapp.com/api

Query Parameter

/[name]

Response

Key Type Description Example
name string Name of orbiter vehicle "Discovery"
serialNumber string Serial number "OV-103"
numberOfFlights int Total number of flights 39
numberOfOrbits int Total number of orbits around Earth 5830
firstMission string First mission (Space Transportation System) "STS-41-D"
lastMission string Last mission (Space Transportation System) "STS-133"
firstLaunchDate string Date of first launch in Month D, Yr format "August 30, 1984"
firstLandingDate string Date of first landing in Month D, Yr format "September 5, 1984"
lastLaunchDate string Date of last launch in Month D, Yr format "February 24, 2011"
lastLandingDate string Date of last landing in Month D, Yr format "March 9, 2011"
milesTraveled int Total miles traveled 148221675
flightHours int Total number of hours in flight 8783

Example Query

https://nasa-orbiter-api/herokuapp.com/api/atlantis
